# Calculator Application

This is a simple and responsive web-based calculator application built with HTML, CSS, and JavaScript.  
It supports basic arithmetic operations including addition, subtraction, multiplication, and division.

## Features

- Clean and user-friendly calculator interface
- Supports decimal number input
- Performs:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
- Prevents invalid decimal input (multiple decimal points)
- Handles division by zero with an `Error` display
- Includes a clear (`C`) button to reset calculations

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Structure

- `index.html` - Calculator layout and button structure
- `styles.css` - Styling and responsive UI design
- `script.js` - Calculator logic and state handling

## How to Run

1. Open the project folder.
2. Double-click `index.html` to run in your browser.

No installation or dependencies are required.
